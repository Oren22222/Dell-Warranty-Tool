# Dell-Warranty-Tool
Use this tool to look up Dell ship dates and warranty expiry dates, with the ability to export results to CSV or XLSX.

Setup Instructions
Save the .hta file to your Desktop.
Once saved, double-click the file on your Desktop to launch the application.
Before you can check ship dates or warranty information, you must obtain a Dell API Key and Client Secret.
These can be generated through the Dell TechDirect Portal under the API access/developer section.
Enter your API credentials into the application when prompted.
You can then:
Import a list of Dell Service Tags
Retrieve device ship dates
Retrieve warranty expiry dates
Export the results to CSV or XLSX format
Notes
An active internet connection is required.
Dell API access may require your organization to be enrolled in TechDirect.
Large imports may take several minutes depending on Dell API rate limits.

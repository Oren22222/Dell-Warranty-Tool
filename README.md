# Dell-Warranty-Tool
Use this tool to lookup dell ship date and warranty expire date and have the ability to export to CSV or XLSX

1.) Save the hta file to desktop

2.) Following this, once it is saved to desktop run the file and the application will open

3.) in order to be able to check the ship date/warranty status of any devices you will need to obatin an API key and client secret.
        This can be obtained from the techdirect dell program
